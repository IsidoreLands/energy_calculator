Energy Calculator

This is a simple GitHub Pages site for the Energy Calculator project, based on Energy-Maneuverability (EM) theory. The Python scripts in this repository handle logging, unit conversions, matrix derivations, and a central hub for EM variables.

Scripts Overview

  • em_logger.py: Handles derivation of Set B and Set C.
  • em_logged_units.py: Converts units to standard military formats.
  • em_logger_matrix.py: Derives Set B from aircraft schema and logs.
  • main_em.py: Central hub to call other scripts.

Testing the Scripts

To test, clone the repo locally and run the scripts via Python. See the repository README or script comments for details.
